# ENCONTRO - App React Native

## Informações do Projeto
- **Nome do pacote**: com.encontro.app
- **Email de suporte**: encontrosuport@outlook.com
- **Versão**: 1.0.0

## Como rodar
1. Instale as dependências: `npm install`
2. Inicie o projeto: `npm start`
3. Build com EAS: `eas build --platform android`

## Arquivos incluídos
- App.js - Código principal
- app.json - Configuração Expo
- eas.json - Configuração EAS Build
- package.json - Dependências
